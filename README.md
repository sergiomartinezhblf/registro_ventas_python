# Sistema de Registro de Ventas CLI en Python

Aplicación de línea de comandos (CLI) desarrollada en Python para registrar, almacenar y analizar ventas utilizando archivos CSV.

## Características

- Registrar ventas desde la terminal
- Guardar datos en un archivo CSV
- Visualización del historial de ventas
- Analizar ventas por producto
- Generar reportes en PDF
- Generación de gráfica de ventas por producto
- Integración de gráfica dentro del archivo PDF

## Tecnologías utilizadas

- Python 3
- Matplotlib
- CSV 
- ReportLab (generación de reportes PDF)
- Git
- Visual Studio Code
- WSL

## Requisitos

- Python 3 instalado
- Entorno virtual (venv)

## Instalación

Clonar el repositorio:

git clone
http://github.com/sergimartinezhblf/registro_ventas_python.git

Entrar al directorio:

cd registro_ventas

Crear entorno virtual:

python -m venv venv

Activar entorno virtual:
Linux/WSL

source venv/bin/activate

Windows

venv/Scripts/activate

Instalar dependencias:

pip install -r requirements.txt
